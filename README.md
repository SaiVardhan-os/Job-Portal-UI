# 💼 Job Portal UI

A full-stack **Job Portal** web application built using the **MERN Stack**. This project enables users to browse job listings through a modern and responsive interface while utilizing a scalable backend powered by Node.js, Express.js, and MongoDB.

---

## 🚀 Features

- 🔍 Browse and search job listings
- ⚡ Fast and responsive user interface
- 🌐 RESTful API integration
- 🗄️ MongoDB Atlas database
- 📱 Responsive design
- 🔒 Environment variable support

---

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/SaiVardhan-os/Job-Portal-UI.git
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Install Frontend Dependencies

```bash
cd ../frontend
npm install
```

---

## ▶️ Run the Application

### Start Backend

```bash
cd backend
npm start
```

### Start Frontend

```bash
cd frontend
npm run dev
```
